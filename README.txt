Muse : Rule Engine
---------------

Implementation of four strategies for rules
 - Basic validation support (see com.msi.brules.BasicRuleExecutor)
 - Simple validation support (see com.msi.brules.SimpleRuleExecutor)
 - Forward chain Production Rule support (see com.msi.brules.ProductionRuleExecutor)
 - Forward chain Production Rule support with parallel validation based on actor model 
 	(see com.msi.brules.ProductionRuleExecutorParallel)
